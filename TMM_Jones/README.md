This directory contains the Python files from the TMM project before refactoring.

## Archived Files

- **`Jones_Xect.py`** - Energy dispersion calculation with Jones matrix
- **`Jones_disperion.py`** - Energy cross-section dispersion calculation with Jones matrix 
- **`Jones_3D_Calculator.py`** - combination of Xect and dispersion
- **`eps_convert_from_ref-web.py`** - Original material property conversion utility
